Scripts

Various *nix scripts I have written for automating photography related tasks, including raw development, film simulation, and image backup.

copyimages — a script for backing up images from sd to external hard drive with date-based folder structure.

film_sim_scripts — a collection of scripts for simulating film photography via the automated application of LUTs with a simple GUI or web server.

Fujixpro1chrome — a DCP profile for applying a Classic Chrome-style tone to X-Pro1 raws. Old and largely obsolete, left up for anyone who finds it useful

OM_TG7_Scripts — a collection of scripts for developing raw files from the OM TG-7, a compact camera with a small sensor in tough body.

termux-camera-to-kindle — a script for transferring photos from Termux to a jailbroken Kindle. Written during covid lockdown for my masters degree.
